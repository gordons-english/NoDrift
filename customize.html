User communication profile

Personalize how Codex works with you.

Configure how Codex communicates, leads work, asks questions, explains decisions, and reports evidence without changing what Codex must verify.

Token savings should come only from communication-style choices, never from reducing verification, source fidelity, privacy protection, approval gates, or external-action safeguards.

Workflow settings

How the work moves

These choices shape who leads, when Codex asks, and how approval stays visible.

1

Working Profile

Choose a preset starting point. This moves the settings below.

If you change any setting manually, the current profile becomes Custom.

2

Work Leadership

Controls who identifies the next task and how Codex moves through approved work.

Approved Task Queue means Codex works through a list you already approved and stops at new boundaries, risks, unclear decisions, or external actions.

3

Question Handling

Controls how Codex handles small decisions during work.

Continue If Clear means Codex keeps working only when the next step is already approved, low-risk, and obvious from the task.

Communication settings

What affects estimated token use

The estimate is based on these communication choices. It is not a guarantee, because real token use still depends on the task and source material.

4

Communication Detail

Controls how much Codex includes beyond the direct answer.

Moving right adds more context, reasoning, examples, tradeoffs, risks, and next-step explanation.

5

Language Level

Controls how technical Codex sounds when explaining work, files, commands, decisions, risks, or next steps.

Moving right uses more precise terminology and assumes more comfort with technical language.

6

Progress Updates

Controls how much Codex reports during longer work.

Moving right shows more of what Codex checked, changed, found, verified, and still needs from you.

7

Explanation Level

Controls how much Codex explains while working.

Moving right adds more explanation, instruction, and advanced context while Codex does the work.

8

Evidence Status

Controls how visibly Codex marks whether information is verified, user-reported, assumed, not inspected, unknown, or needs confirmation.

Moving right gives more visible evidence labels and more explanation of why the status matters.
